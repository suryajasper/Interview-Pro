# Interview-Pro

## Setup Instructions
1. Clone Repository
```bash
git clone https://github.com/suryajasper/Interview-Pro.git
cd Interview-Pro
```
2. Set up and run server: 
```bash
cd web/server
npm install
npm start
```
3. Set up and run client:
```bash
cd web/client
npm install
npm start
# navigate to http://localhost:3000 on Chrome
```
